## Why Each Choice Was Made

- Metadata aliases were added instead of replacing old columns to avoid breaking existing web candidate CSVs and tests.
- Provider adapters validate configuration first because this prompt is about safe source acquisition readiness, not bulk collection.
- Google is implemented only as manual CSV import because scraping Google HTML is explicitly forbidden and fragile.
- Wikimedia is treated as ready without an API key because its public API can be queried without credentials.
- Pexels, Unsplash, and Flickr fail clearly without credentials so future runs are easy to diagnose.
- Bulk downloads were not added because the next milestone is a small approved pilot CSV and review pack.
