## Why Each Variant Was Chosen

- ResNet18 + Mahalanobis is the reference baseline and establishes continuity with the first anomaly result.
- ResNet18 + kNN tests whether local neighbor density works better than one Gaussian normal distribution.
- ResNet18 threshold sweep tests whether the first result was mainly threshold-policy limited.
- ResNet50 + Mahalanobis tests a stronger frozen CNN with the original scorer.
- ResNet50 + kNN combines the stronger feature extractor with the more flexible scorer.
- ResNet18 patch-grid kNN tests whether a simple local-feature method catches defects diluted by global pooling.
