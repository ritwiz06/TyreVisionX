- Feature-map patch kNN tests whether local convolutional descriptors are better than resized patch images.
- The threshold-swept patch variant tests whether patch scores need a more permissive operating point.
- PatchCore-lite tests whether reduced memory and top-k aggregation improve stability without introducing a heavy external framework.
