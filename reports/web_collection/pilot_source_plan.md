# Pilot Source Plan

Updated: 2026-04-19

## Goal

Prepare the first practical source-acquisition pilot for 30-50 tyre candidate images. This is a discovery and review pilot, not a training-data claim.

## Candidate Mix

| View | Target Rows | Notes |
|---|---:|---|
| New tread close-up | 8-12 | Prefer clear normal tread patterns. |
| Clean sidewall close-up | 8-12 | Include readable sidewall texture/lettering. |
| Mounted tyre product photo | 5-8 | Useful for normal whole-tyre context. |
| Inspection-like view | 5-8 | Prefer neutral lighting and minimal background clutter. |
| Industrial/off-highway tyre exterior | 4-6 | Useful variety, but review carefully for dirt/wear. |

## Provider Readiness

| Source | Ready Now | Notes |
|---|---:|---|
| Manufacturer/internal export | yes, if user has data | Best provenance. |
| Manual Google discovery CSV | yes | Manual browser discovery only; no scraping. |
| Wikimedia Commons | yes for API config/check | No API key required, but relevance may be noisy. |
| Pexels | pending API key | Requires `PEXELS_API_KEY`. |
| Unsplash | pending API key | Requires `UNSPLASH_ACCESS_KEY`. |
| Flickr | pending API key | Requires `FLICKR_API_KEY`. |

## Do Not Bulk Download Yet

The first milestone is a small approved CSV and review pack. Bulk downloads should wait until the pilot proves that source metadata, filtering, review, and promotion rules work.
